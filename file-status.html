File Status

You can track the progress of your assessment online. This system provides the most up-to-date information about the status of your file. The portal is updated when documents are received, if a communication is sent, when the application has been reviewed and placed in the queue for assessment, when the assessment in progress, and when the Assessment Report is complete. You will need your date of birth and ICAS File Number to log in. Please note that the file status does not contain any personal information and cannot be used to update your application or add services. The average report processing times posted on ICAS' website are current and up-to-date. Your patience and understanding is appreciated.
